要有HTTP requests的功能
1. 加入dependencies:
pubspec.yaml
```dart
dependencies:
  http: ^1.3.0 
```
2. Depend on it
```shell
dart pub add http
```
3. Import it
```dart
import 'package:http/http.dart';
```
